# `TensorBoardLauncher.py`

## Visao geral
`TensorBoardLauncher.py` inicia o TensorBoard apontando para o diretorio de logs do treinamento.

## Responsabilidades
- Configurar e iniciar o TensorBoard.
- Manter o processo ativo ate o usuario encerrar.

## Funcoes
### `launch_tensorboard(logdir, port)`
Cria e inicia o TensorBoard com o diretorio e porta informados.

## Observacoes
- Usado como utilitario rapido para visualizar metricas.
