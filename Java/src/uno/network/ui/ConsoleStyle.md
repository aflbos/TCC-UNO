# `ConsoleStyle`

## Visao geral
`ConsoleStyle` centraliza codigos ANSI para colorir a saida do console.

## Responsabilidades
- Definir constantes de cores e estilos.
- Fornecer uma camada unica para formatacao de texto.

## Componentes
- `RESET`, `BOLD`, `CYAN`, `GREEN`, `YELLOW`, `RED`, `BLUE`, `MAGENTA`.

## Observacoes
- As cores estao sempre habilitadas no projeto atual.
