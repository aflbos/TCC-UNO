# `RuleSpec`

## Visao geral
`RuleSpec` define conjuntos de regras para treinamentos e avaliacoes de IA. Ela centraliza combinacoes de regras e presets usados no projeto.

## Responsabilidades
- Descrever configuracoes de regras do jogo.
- Fornecer presets reutilizaveis.

## Uso tipico
- Avaliacoes e treinamentos consultam `RuleSpec` para configurar partidas consistentes.

## Observacoes
- O indice das regras segue a ordem usada pela `Simulation`.
